API reference¶

SFINCS model class¶

Initialize¶

sfincs.SfincsModel([root, mode, config_fn, …])

The SFINCS model class (SfincsModel) contains methods to read, write, setup and edit SFINCS models.

Build components¶

sfincs.SfincsModel.setup_config(**cfdict)

Update config with a dictionary

sfincs.SfincsModel.setup_basemaps(region[, …])

Define model region and setup combined bathymetry/elevation and mask static model layers.

sfincs.SfincsModel.setup_river_inflow([…])

Setup river inflow (source) points where a river enters the model domain.

sfincs.SfincsModel.setup_river_outflow([…])

Setup river outflow boundary (msk=3) where a river flows out of the model domain.

sfincs.SfincsModel.setup_gauges([gauges_fn])

Setup model observation point locations.

sfincs.SfincsModel.setup_manning_roughness([…])

Setup model curve number map from gridded curve number map.

sfincs.SfincsModel.setup_cn_infiltration([…])

Setup model curve number map from gridded curve number map.

sfincs.SfincsModel.setup_h_forcing([…])

Setup waterlevel boundary point locations and time series.

sfincs.SfincsModel.setup_q_forcing([…])

Setup discharge boundary based on point location time series.

sfincs.SfincsModel.setup_q_forcing_from_grid([…])

Setup discharge boundary based on gridded discharge data and pre-set river inflow locations, for instance by using the set_rivers() method.

sfincs.SfincsModel.setup_p_forcing_gridded([…])

Setup gridded precipitation forcing.

Plot methods¶

sfincs.SfincsModel.plot_basemap([fn_out, …])

sfincs.SfincsModel.plot_forcing([fn_out])

Attributes¶

sfincs.SfincsModel.region

Returns geometry of region of the model area of interest.

sfincs.SfincsModel.crs

Returns coordinate reference system embedded in staticmaps.

sfincs.SfincsModel.res

Returns coordinates of staticmaps.

sfincs.SfincsModel.root

Path to model folder.

sfincs.SfincsModel.config

Returns parsed model configuration.

sfincs.SfincsModel.staticmaps

xarray.Dataset representation of all static parameter maps

sfincs.SfincsModel.staticgeoms

geopandas.GeoDataFrame representation of all model geometries

sfincs.SfincsModel.forcing

dict of xarray.dataarray representation of all forcing

High level methods¶

sfincs.SfincsModel.read()

Method to read the complete model schematization and configuration from file.

sfincs.SfincsModel.write()

Method to write the complete model schematization and configuration to file.

sfincs.SfincsModel.build(region[, res, …])

Single method to setup and write a full model schematization and configuration from scratch

sfincs.SfincsModel.set_root(root[, mode])

Initialized the model root.

General methods¶

sfincs.SfincsModel.setup_config(**cfdict)

Update config with a dictionary

sfincs.SfincsModel.get_config(*args[, …])

Get a config value at key(s).

sfincs.SfincsModel.set_config(*args)

Update the config dictionary at key(s) with values.

sfincs.SfincsModel.read_config([config_fn])

Parse config from file.

sfincs.SfincsModel.write_config([…])

Write config to <root/config_fn>

sfincs.SfincsModel.set_staticmaps(data[, name])

Add data to staticmaps.

sfincs.SfincsModel.read_staticmaps([crs, nodata])

Read and SFNCS binary staticmaps

sfincs.SfincsModel.write_staticmaps([write_figs])

Write binary SFINCS staticmaps.

sfincs.SfincsModel.set_staticgeoms(geom, name)

Add geom to staticmaps

sfincs.SfincsModel.read_staticgeoms()

Read bnd/src/obs SFINCS xy files If other geojson files are present gis folder, read those as well.

sfincs.SfincsModel.write_staticgeoms()

Write bnd/src/obs to SFINCS xy files If write_gis, write all staticgeoms to geojson files in gis subfolder

sfincs.SfincsModel.set_forcing(data[, name])

Add data to forcing attribute which is a dictionary of xarray.DataArray.

sfincs.SfincsModel.read_forcing()

Read bzs/dis forcing timeseries and netampr gridded forcing

sfincs.SfincsModel.write_forcing()

write bzs/dis 1D forcing and netampr 2D forcing files
