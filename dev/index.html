HydroMT plugin: SFINCS

HydroMT is a python package, developed by Deltares, to build and analyse hydro models. It provides a generic model api with attributes to access the model schematization, (dynamic) forcing data, results and states.

This plugin provides an implementation for the SFINCS hydrological modelling framework. It details the different steps and explains how to use HydroMT to easily get started and work on your own sfincs model.

For detailed information on HydroMT itself, you can visit the core documentation.
