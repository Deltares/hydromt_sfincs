hydromt_sfincs.workflows.landuse

hydromt_sfincs.workflows.landuse(da, ds_like, fn_map, logger=<Logger hydromt_sfincs.workflows.landuse (WARNING)>, params=None)[source]

Returns landuse map and related parameter maps. The parameter maps are prepared based on landuse map and mapping table as provided in the generic data folder of hydromt.

Parameters

  • da (xarray.DataArray) – DataArray containing LULC classes.

  • ds_like (xarray.DataArray) – Dataset at model resolution.

  • fn_map (str, Path) – path to mapping file

Returns

ds_out – Dataset containing gridded landuse based maps

Return type

xarray.Dataset
