hydromt_sfincs.SfincsModel.set_results¶

SfincsModel.set_results(data, name=None)¶

Add data to results attribute which is a dictionary of xarray.DataArray. The dictionary key is taken from the variable name. In case of a DataArray without name, the name can be passed using the optional name argument.

Parameters

  • data (xarray.Dataset or xarray.DataArray) – New forcing data to add

  • name (str, optional) – Variable name, only in case data is of type DataArray
