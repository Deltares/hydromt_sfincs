hydromt_sfincs.utils.mask_bounds¶

hydromt_sfincs.utils.mask_bounds(da_mask: xarray.core.dataarray.DataArray, gdf_include: Optional[geopandas.geodataframe.GeoDataFrame] = None, gdf_exclude: Optional[geopandas.geodataframe.GeoDataFrame] = None, structure: numpy.ndarray = array([[1., 1., 1.], [1., 1., 1.], [1., 1., 1.]])) → xarray.core.dataarray.DataArray[source]¶

Returns the boundary cells based of values greater than zero in da_mask. If provided the bounds are limited to cells within gdf_include and outside gdf_exclude.

Parameters

  • da_mask (xr.DataArray) – Model mask.

  • gdf_include (geopandas.GeoDataFrame) – Geometries with areas to include/exclude from the model boundary.

  • gdf_exclude (geopandas.GeoDataFrame) – Geometries with areas to include/exclude from the model boundary.

Returns

bounds – Boolean mask of model boundary cells.

Return type

xr.DataArray
