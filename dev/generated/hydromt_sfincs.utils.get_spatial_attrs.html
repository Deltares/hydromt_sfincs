hydromt_sfincs.utils.get_spatial_attrs

hydromt_sfincs.utils.get_spatial_attrs(config: Dict, crs: Optional[Union[int, pyproj.crs.crs.CRS]] = None, logger=<Logger hydromt_sfincs.utils (WARNING)>)[source]

Returns geospatial attributes shape, crs and transform from config dict.

The config dict should contain the following keys:

  • for shape: mmax, nmax

  • for crs: epsg

  • for transform: dx, dy, x0, y0, (rotation)

NOTE: Rotation != 0 is not yet supported.

Parameters

  • config (Dict) – sfincs.inp configuration

  • crs (int, CRS) – Coordinate reference system

Returns

  • shape (tuple of int) – width, height

  • transform (Affine.transform) – Geospatial transform

  • crs (pyproj.CRS, None) – Coordinate reference system
