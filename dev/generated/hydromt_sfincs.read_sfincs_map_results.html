hydromt_sfincs.read_sfincs_map_results

hydromt_sfincs.read_sfincs_map_results(fn_map: Union[str, pathlib.Path], hmin: float = 0.0, crs: Optional[Union[int, pyproj.crs.crs.CRS]] = None, chunksize: int = 100, drop: List[str] = ['crs', 'sfincsgrid'], logger=<Logger hydromt_sfincs.utils (WARNING)>, **kwargs) → Tuple[xarray.core.dataset.Dataset][source]

Read sfincs_map.nc staggered grid netcdf files and parse to two hydromt.RasterDataset objects: one with face and one with edge variables.

Additionally, hmax is computed from zsmax and zb if present.

Parameters

  • fn_map (str, Path) – Path to sfincs_map.nc file

  • hmin (float, optional) – Minimum water depth to consider in hmax map, i.e. cells with lower depth get a nodata values assigned. By deafult 0.0

  • crs (int, CRS) – Coordinate reference system

  • chunksize (int, optional) – chunk size along time dimension, by default 100

  • drop (List[str], optional) – Variables to drop from reading, by default [“crs”, “sfincsgrid”]

Returns

ds_face, ds_edge – Parsed SFINCS output map file

Return type

hydromt.RasterDataset
