Running a model#

SFINCS can be run on multiple different platforms, both local, HPC and cloud based. The simplest way is to run SFINCS on Windows using a batch-file, which is shown in this example. The example consists of a simple compound flood model that has been created with HydroMT-SFINCS. The model is situated in Northern Italy and is forced with waterlevel and discharge boundaries.

If you want to read more about running SFINCS on different platforms, please read the [SFINCS manual](https://sfincs.readthedocs.io/en/latest/running.html).
