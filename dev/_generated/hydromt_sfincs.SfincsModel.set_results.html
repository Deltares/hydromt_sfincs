hydromt_sfincs.SfincsModel.set_results#

SfincsModel.set_results(data: DataArray | Dataset, name: str | None = None, split_dataset: bool | None = False)#

Add data to results attribute.

Dataset can either be added as is (default) or split into several DataArrays using the split_dataset argument.

Parameters:

  • data (xarray.Dataset or xarray.DataArray) – New forcing data to add

  • name (str, optional) – Results name, required if data is xarray.Dataset and split_dataset=False.

  • split_dataset (bool, optional) – If True (False by default), split a Dataset to store each variable as a DataArray.
